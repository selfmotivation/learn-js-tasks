Проверка класса: "instanceof"

Задачи

Странный instanceof

Почему instanceof в примере ниже возвращает true? Мы же видим, что a не создан с помощью B().

  1. function A() {}
  2. function B() {}
  4. A.prototype = B.prototype = {};
  6. let a = new A();
  8. alert( a instanceof B ); // true

Решение:

  instanceof при проверке учитывает только prototype, который проверяется на совпадения
  в прототипной цепочке. Здесь a.__proto__ == B.prototype, поэтому instanceof возвращает true.
