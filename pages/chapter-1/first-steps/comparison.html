Операторы сравнения

Каким будет результат этих выражений?

  1. 5 > 4
  2. "ананас" > "яблоко"
  3. "2" > "12"
  4. undefined == null
  5. undefined === null
  6. null == "\n0\n"
  7. null === +"\n0\n"

Решение:

  5 > 4 // true
  "ананас" > "яблоко" // false
  "2" > "12" // true
  undefined == null // true
  undefined === null // false
  null == "\n0\n" // false
  null === +"\n0\n" // false
