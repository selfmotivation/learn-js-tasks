Конструкция "switch"

Задачи

Напишите "if", аналогичный "switch"

Напишите if..else, соответствующий следующему switch:

  1. switch (browser) {
  2.   case 'Edge':
  3.       alert( "You've got the Edge!" );
  4.       break;
  6.   case 'Chrome':
  7.   case 'Firefox':
  8.   case 'Safari':
  9.   case 'Opera':
  10.     alert( 'Okay we support these browsers too' );
  11.     break;
  13.   default:
  14.     alert( 'We hope that this page looks ok!' );
  15. }

Решение:

  if (browser === 'Edge') {
    alert( "You've got the Edge!" );
  } else if (browser === 'Chrome' ||
             browser === 'Firefox' ||
             browser === 'Safari' ||
             browser === 'Opera') {
    alert( 'Okay we support these browsers too' );
  } else {
    alert( 'We hope that this page looks ok!' );
  }
Переписать условия "if" на "switch"

Перепишите код с использованием одной конструкции switch:

  1. const number = +prompt('Введите число между 0 и 3', '');
  3. if (number === 0) {
  4.   alert('Вы ввели число 0');
  5. }
  7. if (number === 1) {
  8.   alert('Вы ввели число 1');
  9. }
  11. if (number === 2 || number === 3) {
  12.   alert('Вы ввели число 2, а может и 3');
  13. }

Решение:

  const number = +prompt('Введите число между 0 и 3', '');
    switch (number) {
      case 0:
        alert('Вы ввели число 0');
        break;
      case 1:
        alert('Вы ввели число 1');
        break;
      case 2:
      case 3:
        alert('Вы ввели число 2, а может и 3');
        break;
    }
