Конструкторы, создание объектов через "new"

Задачи

Две функции - один объект

Возможно ли создать функции A и B в примере ниже, где объекты равны new A()==new B()?

  1. function A() { ... }
  2. function B() { ... }
  4. let a = new A;
  5. let b = new B;
  7. alert( a == b ); // true

Если да – приведите пример вашего кода.

Решение:

  const randomObj = {};

  function A() { return randomObj; }
  function B() { return randomObj; }
    
  alert( new A() === new B() ); // true
Создание калькулятора при помощи конструктора

Создайте функцию-конструктор Calculator, который создаёт объекты с тремя методами:

  • read() запрашивает два значения при помощи prompt и сохраняет их значение в свойствах объекта.
  • sum() возвращает сумму введённых свойств.
  • mul() возвращает произведение введённых свойств.

  1. let calculator = new Calculator();
  2. let calculator = new Calculator(); calculator.read();
  4. alert( "Sum=" + calculator.sum() );
  5. alert( "Mul=" + calculator.mul() );

Решение:

  function Calculator() {
    this.read = function() {
      this.x = +prompt('1st number?', '');
      this.y = +prompt('2nd number?', '');
    },
    this.sum = function() {
      return this.x + this.y;
    },
    this.mul = function() {
      return this.x * this.y;
    };
  }
Создаём Accumulator

Напишите функцию-конструктор Accumulator(startingValue).

Объект, который она создаёт, должен уметь следующее:

  • Хранить «текущее значение» в свойстве value. Начальное значение устанавливается в аргументе конструктора startingValue.
  • Метод read() использует prompt для получения числа и прибавляет его к свойству value.

Таким образом, свойство value является текущей суммой всего, что ввёл пользователь при вызовах метода read(), с учётом начального значения startingValue.

Ниже вы можете посмотреть работу кода:

  1. let accumulator = new Accumulator(1); // начальное значение 1
  3. accumulator.read(); // прибавит ввод prompt к текущему значению
  4. accumulator.read(); // прибавит ввод prompt к текущему значению
  6. alert(accumulator.value); // выведет сумму этих значений

Решение:

  function Accumulator(startingValue) {
    this.value = startingValue;
    this.read = function() {
      this.value = this.value + +prompt('Введите число', '');
    }
  }
