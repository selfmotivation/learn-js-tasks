Навигация по DOM-элементам

Задачи

Дочерние элементы в DOM

Для страницы:

  1. ‹html›
  2. ‹body›
  3.   ‹div›Пользователи:‹/div›
  4.   ‹ul›
  5.     ‹li›Джон‹/li›
  6.     ‹li›Пит‹/li›
  7.   ‹/ul›
  8. ‹/body›
  9. ‹/html›

Напишите код, как получить…

  • элемент ‹div›?
  • ‹ul›?
  • второй ‹li› (с именем Пит)?

Решение:

  1. document.body.firstElementChild;
  2. document.body.lastElementChild;
  3. document.body.lastElementChild.lastElementChild;
Вопрос о соседях

Если elem – произвольный узел DOM-элемента…

  • Правда, что elem.lastChild.nextSibling всегда равен null?
  • Правда, что elem.children[0].previousSibling всегда равен null ?

Решение:

  1. elem.lastChild всегда последний, правда;
  2. elem.children[0] – потомок-элемент, перед ним могут быть другие узлы - не правда;
Выделите ячейки по диагонали

Напишите код, который выделит красным цветом все ячейки в таблице по диагонали.

Вам нужно получить из таблицы ‹table› все диагональные ‹td› и выделить их, используя код:

  1. // в переменной td находится DOM-элемент для тега
  2. td.style.backgroundColor = 'red';

Должно получиться так:

Решение:

  const table = document.body.firstElementChild;

    for (let i = 0; i < table.rows.length; i++) {
      let row = table.rows[i];
      row.cells[i].style.backgroundColor = 'red';
    }
