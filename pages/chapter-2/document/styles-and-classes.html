Задачи

Создать уведомление

Напишите функцию showNotification(options), которая создаёт уведомление: ‹div class="notification"› с заданным содержимым. Уведомление должно автоматически исчезнуть через 1,5 секунды.

Пример объекта options:

  1. // показывает элемент с текстом "Hello" рядом с правой верхней частью окна.
  2. showNotification({
  3.   top: 10, // 10px от верхней границы окна (по умолчанию 0px)
  4.   right: 10, // 10px от правого края окна (по умолчанию 0px)
  5.   html: "Hello!", // HTML-уведомление
  6.   className: "welcome" // дополнительный класс для div (необязательно)
  7. });

Используйте CSS-позиционирование для отображения элемента в заданных координатах. Исходный документ имеет необходимые стили.

Решение:

  function showNotification({top = 0, right = 0, className, html}) {

    const notification = document.createElement('div');
    notification.className = "notification";
    if (className) {
      notification.classList.add(className);
    }

    notification.style.top = top + 'px';
    notification.style.right = right + 'px';

    notification.innerHTML = html;
    document.body.append(notification);

    setTimeout(() => notification.remove(), 1500);
  }
